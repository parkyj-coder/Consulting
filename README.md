# 프리미엄 컨설팅 - 전문적인 비즈니스 솔루션

## 프로젝트 개요
프리미엄 컨설팅은 전문적인 비즈니스 솔루션을 제공하는 웹사이트입니다. 기업의 성장과 발전을 위한 맞춤형 컨설팅 서비스를 제공합니다.

## 기술 스택
- **Frontend**: HTML5, CSS3, JavaScript
- **Backend**: Java, JSP, Servlet
- **Server**: Apache Tomcat 10.1
- **IDE**: Eclipse IDE
- **Version Control**: Git

## 주요 기능
- 🏢 **기업 소개**: 회사 개요 및 비전 소개
- 📊 **서비스 소개**: 제공하는 컨설팅 서비스 안내
- 👥 **팀 소개**: 전문 컨설턴트 팀 소개
- 📞 **연락처**: 고객 문의 및 상담 신청
- 💼 **채용 정보**: 인재 채용 정보 제공

## 설치 및 실행 방법

### 1. 환경 요구사항
- Java 17 이상
- Apache Tomcat 10.1
- Eclipse IDE (선택사항)

### 2. 프로젝트 실행
1. Eclipse에서 프로젝트 Import
2. Tomcat 서버에 프로젝트 배포
3. 브라우저에서 `http://localhost:8080/consulting` 접속

### 3. 정적 파일로 실행
1. `src/main/webapp` 폴더의 내용을 웹 서버에 복사
2. `index.html` 파일을 브라우저에서 직접 열기
